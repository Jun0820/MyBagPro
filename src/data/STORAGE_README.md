# User Data Storage and Persistence

本アプリケーションはサーバーレス（バックエンドなし）で動作するため、ユーザーのデータは以下の方法で保存・蓄積されます。

## 1. 保存場所
- **ブラウザの LocalStorage**: 
  - ユーザー情報、プロフィール、MyBag（セッティング）データはすべてブラウザ固有の領域に保存されます。
  - ページをリロードしたり、ブラウザを閉じても、同じデバイス・ブラウザであればデータは保持されます。

## 2. データの永続化
- `src/context/DiagnosisContext.tsx` にて、状態が変化するたびにオートセーブが実行されます。
- Phase 6の実装により、マイページからデータを **JSONファイルとしてエクスポート（バックアップ）** する機能が追加されます。これにより、万が一ブラウザのキャッシュをクリアしても、ファイルを読み込むことでデータを復元可能にします。

## 3. データのディレクトリ（開発・同期用）
- 開発時やデバッグ用のデフォルトデータは `src/data/` ディレクトリ配下に配置されています。
- ユーザー個別の動的なデータは、ブラウザ内の `localStorage["mybagpro_profile"]` などのキーに格納されています。
