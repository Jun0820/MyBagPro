# Full Bag Content Checklist

## 目的
- `14本のセッティング` を正しく掲載して、信頼できる閲覧体験を作る
- そのうえで `ドライバー` と `ボール` の診断導線を強くする

## 1人あたり必須
- 名前
- 生年月日
- 身長
- 出身地 / 国籍
- 属性
- HS
- 平均スコア
- ベストスコア
- 使用ボール
- SNS
- 特徴3つ
- 14本のセッティング情報

## 14本で優先して揃える項目
- クラブカテゴリ
- ブランド
- モデル名
- ロフト
- シャフト名
- フレックス
- キャリー飛距離
- トータル飛距離
- いつ時点のセッティングか
- 参照元URL

## ドライバーとボールで特に厳密に揃える項目
- ブランド
- モデル名
- ロフト
- シャフト名
- フレックス
- キャリー飛距離
- トータル飛距離
- いつ時点のセッティングか
- 参照元URL

## 後からでもよい項目
- FW / UT / Iron / Wedge / Putter の細かな飛距離
- ボール変更履歴
- コメント動画の抜粋
- 写真アセット

## 初期の優先掲載順
1. トップスター
2. YouTube・人気
3. 実力派・賞金上位
4. 実力者（50名）
5. ベテラン・伝説

## 決定済みルール
- 最初に公開する10名は `launch-10-seed-plan.md` に固定
- 掲載情報の参照元は `公式 + YouTube + Instagram` を許容
- 「最新」の優先順位は `2026シーズン基準 > 直近試合基準 > 代表セッティング基準`
- 平均スコアが不明な場合は空欄
- HSが不明な場合は空欄
- 14本確認できていないプロフィールは公開しない
